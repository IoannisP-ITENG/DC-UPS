# Modular DC UPS builder

This project shall be able to generate hardware designs for DC uniterruptabe power supplies (UPS).

It takes parameters like 
- input_voltage_min_V
- input_voltage_max_V
- output_voltage_V
- peak_current_mA
- duration_s
- outage_current_mA  
- cell_preference
- ...

## Architecture
Runs on [faebryk](https://github.com/faebryk/faebryk).
